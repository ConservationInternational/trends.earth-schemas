te_schemas documentation
========================

`te_schemas` is a Python package containing the schemas used by the
Trends.Earth QGIS Plugin and related tools. It contains schemas used for
passing results between the Trends.Earth API server and the QGIS plugin, as
well as schemas that are used for reporting results to the United Nations
Convention to Combat Desertification's PRAIS system.
